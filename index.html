Présence digitale

Un profil est projeté. Tout le groupe scanne le même QR code. Chaque personne donne son avis depuis son téléphone. Les réponses remontent dans le tableau avec la moyenne et les verbatims.

Préparer l’audit

Choisis le profil que le groupe va regarder ensemble, ou saisis un invité.

Mode simple : clique sur “Créer le QR code”. Si Apps Script n’est pas encore à jour, la page passera en mode démo local pour tester les clics.

À projeter

Quand le QR code apparaît, tout le monde scanne. Pour tester seule, clique sur “ouvrir le formulaire participant”.

Le QR code apparaîtra ici après création de l’audit.
