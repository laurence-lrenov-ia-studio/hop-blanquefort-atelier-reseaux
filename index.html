Boost LinkedIn collectif

Un profil projeté, le groupe répond depuis son téléphone, puis le membre repart avec des retours concrets et des idées d’action.

Mode test : le QR code ouvre le formulaire participant. Pour récupérer les réponses de 22 téléphones dans un même Sheet, il faudra ensuite brancher ce formulaire au Google Sheet / Apps Script. Ici, tout fonctionne déjà en local pour valider le parcours.

Déroulé projeté :
1. On affiche le profil LinkedIn à l’écran.
2. Le groupe scanne le QR code.
3. Chaque personne répond sur son téléphone.
4. On affiche la synthèse collective.
Lien à scanner :

Sur fichier local, le QR peut ne pas marcher sur d’autres téléphones. Il marchera correctement une fois le HTML publié sur Netlify / GitHub / cockpit.
